Sistema Interactivo Carlos Fonseca

Este proyecto es un sistema educativo sobre la vida de Carlos Fonseca Amador.  
Incluye:

- Interfaz tipo aplicación con menú y botones
- Agente de IA para responder preguntas
- Quiz interactivo para evaluar conocimientos
- Línea de tiempo histórica
- Todo en HTML, CSS y JS
 Cómo usar

1. Abre el enlace de GitHub Pages:
https://norvingarci09-debug.github.io/Sistema-Carlos
2. Navega por las secciones y responde el quiz.
